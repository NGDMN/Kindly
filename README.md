# Kindly

Plataforma de voluntariado gamificada


## Stack

- **Frontend:** React + Bootstrap
- **Backend principal:** Java + Spring Boot
- **Módulo de pontuação:** Python + Flask + Pandas
- **Banco de dados:** Oracle SQL (FIAP)
